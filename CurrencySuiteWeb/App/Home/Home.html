Currency Converter

Select a cell with the currency which you want to be converted

Select a date to use the conversion rate from that particular day

Select the required currencies from and to categories from the drop downs or else use the currency codes displayed in the drop downs

Example: "100 USD EUR" converts 100 US Dollars to Euro

To select a date using the currency codes type the date after the codes

Example: "100 USD EUR 10/4/2001" converts 100 US Dollars to Euro on the 10th of April 2001
