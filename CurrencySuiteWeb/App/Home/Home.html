Welcome to the Currency Suite Quick Tutorial

There are three ways of performing a conversion

1: Type any number (eg. "100"). This will convert the typed value from the currency in the top drop down, to the currency in the bottom drop down

2: Type any number followed by two currency codes (eg. "100 USD EUR"). This will convert the typed value from the first currency code to the second currency code

3: Type any number followed by two currency codes and a date in the format dd-mm-yyyy (eg. "100 USD EUR 28-02-2014"). This will convert the typed value from the first currency code to the second currency code based on the date specified by your query

And there you have it, with this you're ready to get started converting!
